# Shape Utility Classes
Single-purpose, consistent, and predictable classes for styling/modifying an element's shape.

## Radius
**Syntax:** `r[direction]-[size]`
```scss
// All sides
.r-xs { ... }
.r-sm { ... }
.r-md { ... }
.r-lg { ... }
.r-xl { ... }
.r-pill { ... }

// Top
.rt-xs { ... }
.rt-sm { ... }
.rt-md { ... }
.rt-lg { ... }
.rt-xl { ... }
.rt-pill { ... }

// Bottom
.rb-xs { ... }
.rb-sm { ... }
.rb-md { ... }
.rb-lg { ... }
.rb-xl { ... }
.rb-pill { ... }

// Left
.rl-xs { ... }
.rl-sm { ... }
.rl-md { ... }
.rl-lg { ... }
.rl-xl { ... }
.rl-pill { ... }

// Right
.rr-xs { ... }
.rr-sm { ... }
.rr-md { ... }
.rr-lg { ... }
.rr-xl { ... }
.rr-pill { ... }
```
